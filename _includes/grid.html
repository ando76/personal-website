Grid

Notes

The grid object is a container for grid__items. grid__items simply flow one after the other as they are set to float:left. The grid object helps to keep things in place (i.e. avoiding grid__items jumping up onto a previous row that perhaps doesn't total 100%) so a new grid should be started for each row of grid__items.
